JAVASCRIPT

En el año 1995 en el proyecto para el navegador Netscape Navigator con el objetivo de hacer la web más dinamica e interactiva.

JavaScript posee características de lenguajes funcionales y, de hecho, define funciones como elementos de primera clase, pero no es un lenguaje funcional puro.

En 1996, Microsoft creó, el lenguaje jScript con características precisas y hizo que los sitios desarrollados con una de las dos versiones del lenguaje no funcionen.

Chrome, permitió tener aplicaciones complejas en ejecución en el navegador, es decir, pruduce el renacer de JavaScript.


Los nuevos procesadores multicore o multinúcleo permiten la ejecución real en paralelo de los programas.

La creación de un estandar de JavaScript aceptado por todos los fabricantes de navegadores. Este estandar se conoce como ECMAScript.

Todos estos acontecimientos han determinado un renovado y creciente interés por JavaScript, incluso fuera de su entorno tradicional (el web).

Actualmente es posible crear aplicaciones stand alone en JavaScript que no necesitan un navegador. Entre estas aplicaciones destacamos:

  • Aplicaciones de escritorio

  • Aplicaciones de servidor gracias a plataformas como Node.js.

  • Aplicaciones Móviles.


  • Aplicaciones IoT.
